Your app has been installed!

Now that the app's all set up in your organization, we need to install it to a repository.

Find the app

1

Navigate to https://github.com/organizations/<org-name>/settings/apps and you should be able to see your newly setup app listed.

Click on the Edit button.

Install the app

2

On the lefthand pane, click Navigate to https://github.com/organizations/<org-name>/settings/apps and you should be able to see your newly installed app listed.

Click on the Edit button.
